# Theme Showcase

Сайт-лаборатория с 60 высококлассными темами интерфейса и расширенными стадиями визуального тестирования.

## Запуск

Открой `index.html` в браузере.

## Структура

- `themes.js` — каталог из 60 тем и дизайн-токены.
- `theme-engine.js` — применение токенов к CSS-переменным.
- `lab.js` — генерация UI Laboratory блоков и интеракций.
- `app.js` — логика выбора темы, валидация количества тем, синхронизация метаданных.
- `style.css` — bento-сетка, kitchen sink-компоненты, слои, виджеты и состояния.
- `THEMES.md` — описание всех тем и правила расширения.

## Стадии тестирования в UI Laboratory

1. Bento Grid Layout.
2. Kitchen Sink UI (buttons + form + status).
3. Layering & Elevation (hover, dropdown, toast).
4. Content Modules (music, chart, chat).
5. Typography Matrix.
6. Media Test с тематическими фильтрами.
7. State Feedback (:focus, :disabled, range/toggle).
