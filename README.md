# Theme Showcase

Сайт-демонстратор с 30 высококлассными темами интерфейса для быстрого выбора визуального направления.

## Запуск

Открой `index.html` в браузере.

## Структура

- `themes.js` — каталог из 30 тем и дизайн-токены.
- `theme-engine.js` — применение токенов к CSS-переменным.
- `app.js` — UI-логика выбора темы и контроль, что в каталоге ровно 30 тем.
- `style.css` — базовая дизайн-система и эффекты.
- `THEMES.md` — описание всех тем и правила расширения.
